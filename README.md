# Customer_Churn_Prediction
# Project Overview
This project predicts whether a telecom customer will churn (leave the service) or not using Machine Learning classification models.
## Technologies Used
Python
Pandas
NumPy
Scikit-learn
Matplotlib
## Algorithms Used
Logistic Regression
Random Forest
Model Accuracy
## Logistic Regression: 78.74%
## Random Forest: 78.53%
## Dataset
Telco Customer Churn Dataset
## How It Works
Data cleaning and preprocessing
Encoding categorical variables
Train-Test split
Model training
Model evaluation using confusion matrix and classification report
Feature importance analysis
